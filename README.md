# Mawashi System

Mawashi is a web-based system designed to manage and track livestock sacrifice orders for companies operating in Saudi Arabia. The platform helps administrators manage orders and allows users to track the progress of their sacrifice using a shipment number.

## Overview

The system provides a complete order lifecycle management process, from selecting the sacrifice type to the final distribution to beneficiaries.

Users can easily track their order status through the website using the shipment number.

Website: https://mawashi.org/

---

## Features

### Admin Dashboard
- Create and manage sacrifice orders
- Update order status in real time
- Track the full lifecycle of each order
- Manage shipment numbers for customer tracking

### Order Tracking
Customers can track their orders through multiple stages including:

1. Selection of Sacrifice Type
2. Payment & Issuance
3. At the Farm
4. On the Way to Warehouse
5. Arrival at Warehouse
6. On the Way to Slaughterhouse
7. Arrival at Slaughterhouse
8. Preparing Sacrifice for Slaughter
9. Sacrifice Completed
10. Cutting and Packaging Preparation
11. Distribution to Beneficiaries
12. Order Completed

---

## System Workflow

The system is designed to reflect the real operational process of livestock sacrifice services in Saudi Arabia, ensuring transparency and clear tracking for customers.
